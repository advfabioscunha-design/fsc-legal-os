# Checklist de Cadastros — FSC Legal OS v4 (na ordem)

Tenha em mãos: celular (verificações por SMS), cartão de crédito, CNPJ/CPF do escritório.

**E-MAIL OFICIAL DOS CADASTROS: `contato@fscadvocaciadigital.com.br`** (Titan/HostGator).
Use este e-mail profissional em TODOS os cadastros abaixo (Asaas, ZapSign,
WhatsApp/Meta, ElevenLabs, BirdID). Exceções já criadas com o Gmail
(adv.fabios.cunha@gmail.com): Google Cloud, GitHub, Supabase, Vercel,
DigitalOcean — podem ser migradas depois, sem urgência.

## 1. Supabase — banco de dados, login e documentos (GRÁTIS)
- Site: https://supabase.com
- Cadastro: "Sign Up" → entrar com a conta GitHub (faça o passo 2 antes) ou e-mail.
- Depois: "New Project" → nome `fsc-legal-os` → região **South America (São Paulo)** → gerar senha forte do banco (guarde).
- ➜ Me chame logado: eu rodo o banco, crio o cofre de documentos e importo suas teses por grupo.

## 2. GitHub — onde o código mora (GRÁTIS)
- Site: https://github.com/signup
- Cadastro com seu e-mail → criar repositório **privado** chamado `fsc-legal-os`.
- ➜ Me chame: eu preparo e subo o código.

## 3. DigitalOcean — o servidor (VPS) do backend e dos robôs (~US$ 24/mês)
- Site: https://www.digitalocean.com (botão Sign Up — pede cartão)
- Depois: "Create → Droplet" → Ubuntu 24.04 → plano Basic **4GB/2vCPU** → região New York ou São Paulo.
- ➜ Me passe o IP do Droplet: eu te guio no comando único de instalação.

## 4. Vercel — hospeda o site/app (GRÁTIS)
- Site: https://vercel.com/signup
- Entrar com a conta GitHub (1 clique) → depois "Import Project" → repositório `fsc-legal-os`, pasta `frontend/`.

## 5. Asaas — cobranças Pix/boleto/cartão (sem mensalidade; taxa por transação)
- Site: https://www.asaas.com
- Cadastro como PJ (CNPJ do escritório) ou PF → validação de identidade (selfie + documento).
- Depois: Menu → Integrações → **API Key** (guarde).

## 6. ZapSign — assinatura digital de contratos (plano a partir de ~R$ 40/mês)
- Site: https://zapsign.com.br
- Cadastro → Configurações → **Integrações/API** → gerar Token.

## 7. WhatsApp Cloud API — o número oficial do escritório (pago por conversa)
- Site: https://developers.facebook.com
- Entrar com sua conta Facebook/Meta → "My Apps" → Create App → tipo **Business** → adicionar produto **WhatsApp**.
- Precisa de um número de telefone dedicado (não pode estar em uso no WhatsApp comum).
- ➜ Esta é a mais chata das configurações: faça só o cadastro e me chame.

## 8. ElevenLabs — a sua voz clonada para os áudios (~US$ 5/mês no início)
- Site: https://elevenlabs.io
- Cadastro → plano Starter → Voices → **Add Voice → Instant Voice Clone** → grave 2–3 min lendo um texto variado, em ambiente silencioso → copie o **Voice ID**.

## 9. BirdID (Soluti) — certificado digital A3 em nuvem (~R$ 35/mês) [pode deixar por último]
- Site: https://birdid.com.br
- Emissão exige videoconferência ou ponto presencial com seus documentos (como todo certificado ICP-Brasil).

---
**Ordem prática recomendada:** 2 (GitHub) → 1 (Supabase) → 3 (DigitalOcean) → 4 (Vercel) → 5/6 (Asaas/ZapSign) → 8 (ElevenLabs) → 7 (WhatsApp) → 9 (BirdID).
Depois de cada cadastro, me chame que eu assumo a configuração na tela.
